# CLO-24: Create Global Settings (Payload Global)

## Overview

Singleton global settings via Payload Global. Site configuration: branding, contact info, social links, policies, tax, currency, payment gateways.

## Acceptance Criteria

- [ ] Payload Global config (not a Collection)
- [ ] Branding: siteName, tagline, logo, favicon
- [ ] Contact: email, phone, address
- [ ] Social links: Instagram, Facebook, YouTube, Pinterest (URLs)
- [ ] Policies: shippingPolicy, returnPolicy (textarea)
- [ ] Tax: gstPercent (default 5), currency (INR)
