# CLO-25: Configure Payload SEO plugin

Configure SEO fields for Products, Pages. Meta title, description, og_image, keywords. Auto-generate fallback from content. Sitemap generation.

- [ ] Install @payloadcms/plugin-seo

- [ ] Add SEO fields to Pages collection
- [ ] Auto-generate meta from title/description fallback
- [ ] Register plugin in payload.config.ts
- [ ] Unit tests for SEO field presence
