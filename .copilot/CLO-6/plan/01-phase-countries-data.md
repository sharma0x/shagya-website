# Phase 1: Data Files

## Steps

### 1.1 Countries Data (`src/lib/countries.ts`)

- Array of country codes + display names
- Countries: India, USA, UK, UAE, Canada, Australia, Singapore, Malaysia, Nepal, Bangladesh, Sri Lanka, Germany, France, Japan, China
- Special "Other" option as last entry
- Export types: `CountryOption`, `COUNTRY_OPTIONS`

### 1.2 India States Data (`src/lib/indian-states.ts`)

- Array of 28 states + 8 UTs with codes and display names
- All standard Indian states (Andhra Pradesh, Arunachal Pradesh, ..., West Bengal)
- All UTs (Andaman & Nicobar, Chandigarh, ..., Ladakh, Puducherry)
- Export types: `IndianState`, `INDIAN_STATES`
