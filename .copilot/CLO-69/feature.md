# CLO-69: Remove strikethrough prices and discount badges

## Overview

Strikethrough prices and red "25% OFF" badges make the brand look like a discount marketplace. Replaced with clean single price + subtle "Special Price" text badge.

## Changes

- Removed red discount percentage pill from ProductCard
- Removed strikethrough compare-at price
- Added "Special Price" text badge below price when product has compareAtPrice > basePrice
