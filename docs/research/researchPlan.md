# Saree E-Commerce Website Research Plan

## Goal
Research 7 top Indian saree websites to understand site architecture, page types, product attributes, e-commerce features, and selling strategies. Document each site in a separate camelCase markdown file in `docs/`.

## Research Protocol (Strictly Follow Per Site)

### Step 1: Site Entry & Navigation
- Fetch homepage → document nav structure (categories, mega-menu, filters)
- What collections/categories are shown?
- Logged-in vs guest experience observation

### Step 2: Category/Listing Page
- How are products listed? (grid/list, items per row)
- Filter options: fabric, type, price, color, occasion, work/embroidery
- Sort options (newest, price, popularity)
- Pagination or infinite scroll
- Visual variations (hover effects, quick view)

### Step 3: Product Detail Page (PDP)
- URL structure pattern
- Image layout (single/zoom/carousel/360-view/video)
- Product attributes displayed:
  - Name, price, MRP, discount %
  - Fabric/Material
  - Length (standard variations)
  - Blouse piece included? (yes/no, unstitched/stitched)
  - Color variants shown
  - Size (free size / custom measurements)
  - Care instructions
  - Availability / stock indicator
- Add to cart / Buy now CTAs
- Reviews & ratings
- Size chart / fit guide
- Share/save/wishlist
- Delivery info, return policy
- COD availability display
- Bundles / combos offered
- Recommended / related products

### Step 4: Attributes Schema
- List every possible attribute for a saree product
- Categorise: identity, physical, pricing, media, inventory, shipping

### Step 5: Features Inventory
- Wishlist / Favourites
- Recently viewed
- Price drop alerts
- Bridal registration / appointment
- Try-on (AR/VR)
- Gift wrapping / gift message
- Bulk / corporate orders
- COD
- EMI options
- International shipping
- Blouse stitching service
- Store locator
- Appointment booking (bridal)
- Chat / WhatsApp support
- Blog / style guide
- Video (drape guide, styling)
- Size customization
- Store pickup

### Step 6: Selling Strategy
- Discounting patterns (seasonal, clearance, new arrival)
- Bundles (saree+blouse, saree+petticoat)
- Loyalty / rewards program
- Referral program
- First order discount
- Free shipping thresholds
- Return policy (length, conditions)
- Payment options breadth
- Cross-sell / upsell on PDP
- Abandoned cart signals
- Email/SMS marketing (newsletter signup popups)
- Urgency tactics (low stock, timer)

### Step 7: Page Types
- Homepage layout & sections
- Category / Collection page
- Product listing page (PLP)
- Product detail page (PDP)
- Cart page
- Checkout page
- Account / Profile page
- Order tracking
- Wishlist page
- Blog / style guide
- Lookbook
- Store locator
- About / Brand story
- Size chart page
- FAQ
- Return / exchange portal
- Contact us
